什么是GitHub Pages?

GitHub Pages是一种静态站点托管服务，直接把你的页面托管到GitHub存储库。

官方说明你可以使用Jekyll Theme Chooser在线创建和发布GitHub Pages网站。当然你可以在本地工作，可以使用GitHub Desktop或命令行。

GitHub Pages不支持服务器端代码，如PHP，Ruby或Python。

（一）注册账号

  1. 访问 github : https://github.com/

  2. 注册账号 :

    输入用户名、E-mail、密码点击 sign up (注意去邮箱验证账号)
    如下三步只要九年义务英语过关就能看懂。

    (二) 创建repository

  3. 登陆进去点New repository,创建新的存储库。

  4. 这里repository name的内容有要求:
    用户名.github.io
    比如我的用户名为yookingwong我就要用yookingwong.github.io

    点击Create repository

    （三） 充实你的库

  5. 现在你的存储库是空的
